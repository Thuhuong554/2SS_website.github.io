## Wireframe: 

<img src="img_for_Readme/Homepage.png" alt="Homepage"/>

We will have 6 main pages:

- **Homepage:** Include all basic information about the bakery
    + Hover view information of product in Best Seller part
    + Pop up items in cart and automatic calculate
    + Hover search page when click search icon
    + Embeded map
  <img src="img_for_Readme/General_wireframe/Homepage.png" alt="Homepage"/>
  <img src="img_for_Readme/General_wireframe/Cake_Infor_hover.png" alt="Cake Infor hover"/>
  <img src="img_for_Readme/General_wireframe/Cartpopup.png" alt="Cartpopup"/>
  <img src="img_for_Readme/General_wireframe/Search_hover.png" alt="Search hover"/>

- **About us**
  <img src="img_for_Readme/General_wireframe/About us.png" alt="About us"/>
- **Menu:**
    + Grid view of product list
    + Have fitler for better searching
    + Have hover view information of each product
    + Have shopping cart for faster buy
    + Have sort by price and name
  <img src="img_for_Readme/General_wireframe/Menu.png" alt="Menu"/>
  <img src="img_for_Readme/General_wireframe/Cake_infor_hover in Menu.png" alt="Cake Infor hover in Menu"/>
- **Cart:**
    + Automatic calculate total final price of each products when add or remove quantity
    + Automatic calculate total cost of all products
    + Automatic calculate final cost after apply promotion code
  <img src="img_for_Readme/General_wireframe/Cart.png" alt="Cart"/>
- **Checkout:**
    + Have bill part for better vision to order
    + Collect information of customer to delivery
    + Thank you page after customer buy successfully
  <img src="img_for_Readme/General_wireframe/Checkout.png" alt="Checkout"/>
  <img src="img_for_Readme/General_wireframe/Thank you.png" alt="Thank you"/>
- **Contact us**
    + Embaded map
    + Form to send customer's ideas
  <img src="img_for_Readme/General_wireframe/Contact.png" alt="Contact us"/>
